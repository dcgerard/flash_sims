First T-FLASH Fit of GTEX Data

David Gerard

2016-04-05

Abstract

I give a first pass at fitting T-FLASH on the GTEX data. I find some interpretable factors. But I also find that I’m mostly interpreting results in terms of tissue and gene factors, and not the individual factors. I wonder if treating samples as independent observations would be more appropriate.

GTEX Data Munging Summary

I only looked at tissues that had at least 20 samples, excluding brain tissues and any sex-dependent tissues (Vagina, Testis, Uterus, Prostate, and Ovary).

For each gene, I calculated the following score:

  1. Normalize by log(counts + 1),
  2. Take the average within each of the saved tissues,
  3. Take the max of the averages and set this to be the gene’s score.

The genes with the top 1000 scores were then saved. The intuition behind this is that genes that are highly expressed within only a couple tissues will still make it into the dataset.

Taking the dataset with only the top 1000 genes, I then normalized each sample by taking log(counts + 1) then subtracting off the 75th percentile.

Expression data from samples of the same individual/tissue combination were then averaged together.

Finally, I only kept subjects that had at least 5 tissues samples. The final data array had dimensions (1000, 496, 21), where the dimensions are genes by individuals by tissues. The proportion of observations that were missing in this dataset was 0.5074. These are structured missing values and not sampled independently, where most individuals are missing whole tissues. If the mean actually is low cp-rank, I don’t think this is a big deal as long as these tissues are missing at random.

panel.cor <- function(x, y, digits = 2, prefix = "", cex.cor, ...)
{
    usr <- par("usr"); on.exit(par(usr))
    par(usr = c(0, 1, 0, 1))
    r <- abs(cor(x, y))
    txt <- format(c(r, 0.123456789), digits = digits)[1]
    txt <- paste0(prefix, txt)
    if(missing(cex.cor)) cex.cor <- 0.8/strwidth(txt)
    text(0.5, 0.5, txt, cex = cex.cor * r)
}

General Comments

I’m not too sure the subjects mode is super important. Mostly, it’s just loading moderately positively. I only start seeing patterns at the fifth factor. It might make more sense to treat subjects as independent replications. For example \[ Y_i \overset{ind}{\sim} N_{p \times n}(F L^T, \sigma^2 I_p \otimes I_n). \]

We could also include a regression term for age and overall mean. For example \[ E[Y_{it}] = FL^T + (\alpha_i + \mu)11^T, \] where \(\alpha_i\) is the main effect for age group \(i\) (\(t\) indexes a replicate here), \(\mu\) is the overall mean, and \(1\) is the vector of ones. This is different from the current model: \[ E[Y_i] = F C_i L^T, \] where \(C_i\) is a diagonal matrix with the samples factors.
