Another Imputation Example

David Gerard

2016-03-30

Abstract

Just one example of where inserting zeros can go wrong.

Generate Data and Include Missing Data in Variational Step

Most of the time, when we run simulations, we generate the true means from some distribution that has mean zero. Of course, imputing zeros will work very well in this situtation. However, if the means do not have mean zero, this can be suboptimal. Here, I generate some rank 1 mean data where the factors and loadings do not have mean zero. I make 60% of the data to be missing. First, I handle the missing data in the appropriate way. The estimates are very good.

library(flashr)
library(ggplot2)
set.seed(31)
n <- 100
p <- 100
u <- rnorm(n, mean = 10)
v <- rnorm(p, mean = 10)
row_cov_half <- diag(sqrt(seq(1, 5, length = n)))
col_cov_half <- diag(sqrt(seq(1, 5, length = p)))
E <- row_cov_half %*% matrix(rnorm(n * p), nrow = n) %*% col_cov_half
Y <- u %*% t(v) + E

pmiss <- 0.6
Omega <- sort(sample(1:(n * p), size = round(pmiss * n * p)))
Y[Omega] <- NA

tout <- tflash_kron(Y = Y)
qplot(u, tout$post_mean[[1]], xlab = "Truth", ylab = "Estimate", main = "Loadings")

qplot(v, tout$post_mean[[2]], xlab = "Truth", ylab = "Estimate", main = "Factors")

qplot(diag(row_cov_half)^2, 1 / tout$sigma_est[[1]], xlab = "Truth",
      ylab = "Estimate", main = "Row Cov")

qplot(diag(col_cov_half)^2, 1 / tout$sigma_est[[2]], xlab = "Truth",
      ylab = "Estimate", main = "Col Cov")

Impute with Zeros

Now I just place zeros where the missing data occur. The estimates are much worse.

Y[Omega] <- 0
tout <- tflash_kron(Y = Y)
qplot(u, tout$post_mean[[1]], xlab = "Truth", ylab = "Estimate", main = "Loadings")

qplot(v, tout$post_mean[[2]], xlab = "Truth", ylab = "Estimate", main = "Factors")

qplot(diag(row_cov_half)^2, 1 / tout$sigma_est[[1]], xlab = "Truth",
      ylab = "Estimate", main = "Row Cov")

qplot(diag(col_cov_half)^2, 1 / tout$sigma_est[[2]], xlab = "Truth",
      ylab = "Estimate", main = "Col Cov")
